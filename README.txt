

# 🦷 Zh.Stom — Стоматологическая CRM система

## 🇷🇺 Описание проекта

Zh.Stom — это современный веб‑сайт стоматологической клиники с интегрированной CRM‑системой для онлайн‑записи пациентов, управления заявками и администрирования.

Проект разработан как полноценная система для стоматологических клиник Казахстана. Основная цель проекта — автоматизация процесса записи пациентов, улучшение взаимодействия между клиникой и клиентами, а также создание современного и удобного интерфейса.

Система позволяет:

- записываться на приём онлайн;
- выбирать врача, дату и время приёма;
- управлять заявками через admin‑панель;
- изменять статусы заявок;
- хранить данные пациентов в базе данных;
- быстро связываться с пациентами через WhatsApp;
- просматривать аналитику заявок.

---

## 🧾 Онлайн‑запись пациентов

Пользователь может:

- указать имя и номер телефона;
- выбрать услугу;
- выбрать врача;
- выбрать дату и время приёма;
- оставить комментарий.

После отправки заявки данные автоматически сохраняются в базе данных SQLite.

---

## 🛠 Admin Panel

В системе реализована полноценная admin‑панель.

Функции admin‑панели:

- просмотр всех заявок;
- изменение статусов;
- удаление заявок;
- WhatsApp‑связь с пациентами;
- просмотр даты и времени записи;
- просмотр выбранного врача;
- аналитика заявок.

Дополнительно реализован скрытый вход в admin‑панель через PIN‑код.

---

# 🧑‍⚕️ Система врачей

В проекте реализована система выбора врача.

Пример:

- Жанара Алимова — стоматолог‑терапевт;
- Арман Сейтов — хирург‑имплантолог;
- Диана Касымова — ортодонт.

Также реализована система выбора даты и времени записи.

# 🗄 База данных

В проекте используется SQLite.

Таблица `appointments` хранит:

- имя пациента;
- телефон;
- услугу;
- врача;
- дату приёма;
- время приёма;
- комментарий;
- статус заявки;
- дату создания.

Также используется таблица `doctors`.

# 🇰🇿 Zh.Stom стоматологиялық CRM жүйесі

## Жобаның сипаттамасы

Zh.Stom — стоматологиялық клиникаға арналған заманауи веб‑сайт және CRM жүйесі.

Жобаның негізгі мақсаты — пациенттерді онлайн қабылдауға жазу процесін автоматтандыру және клиника жұмысын жеңілдету.

Жүйе арқылы:

- онлайн қабылдауға жазылуға;
- дәрігерді таңдауға;
- күн мен уақытты таңдауға;
- өтінімдерді басқаруға;
- пациенттер деректерін сақтауға болады.
